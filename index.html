Heading ou cabeçalho

Os headings, são uma forma de tag utilizada para destacar titulos na pagina, a aumentando seu tamanho. Nesta pagina esta usando os headings h2, mas tem variações de tamanho que vão do h1 até o h6.

Tag de parágrafo

insere um texto formatado como paragrafo padrão. Tem a tag de abertura sendo "p".

Bold ou negrito

A tag bold é utilizado para formatar um texto como negrito. exemplo de texto entre as tags "b".

Italico

A tag italic é utilizado para formatar um texto como italico. exemplo de texto entre as tags "i".

Break ou quebra

A tag "br" serve para quebrar a linha do paragrafo.
